Gamify WooCommerce is a plugin that adds gamification to your WooCommerce store. Reward your customers with points, discounts, and free products to increase engagement and loyalty.

== Description ==

Gamify WooCommerce allows you to reward your customers for their actions on your WooCommerce store. Customers can earn points and redeem them for rewards such as:
- Discount coupons
- Free products
- Free shipping

**Features:**
- Fully integrated with WooCommerce.
- Points system to reward customers for purchases, with other actions such as reviews, and social shares to come.
- Redeem points for coupons, free products, or shipping.
- AJAX-powered user dashboard for redeeming rewards.
- Responsive design for the rewards grid.
- Admin controls for managing rewards and configuring options.
- Secure redemption process with point validation.

**Use Cases:**
- Encourage repeat purchases by offering points for every order.
- Provide exclusive discounts and products for loyal customers.
- Boost customer engagement with gamification elements.

== Installation ==

1. Download the plugin and upload the ZIP file via the WordPress Admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to WooCommerce > Rewards to configure your reward settings and create rewards.

== Frequently Asked Questions ==

= How do I set up rewards? =
Navigate to **WooCommerce > Rewards** to create and manage rewards. You can configure reward types, points required, and specific restrictions like products or categories.

= Can I offer free products as a reward? =
Yes! The plugin supports free product rewards. Customers can redeem points to add a product to their cart at no cost.

= How do customers view and redeem rewards? =
Customers can view and redeem rewards from the **My Account > My Points** page in their user dashboard.

= Is the plugin mobile-friendly? =
Yes, the rewards grid and user dashboard are fully responsive.
